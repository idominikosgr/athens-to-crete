# Athens to Crete

A compact prompt playbook for practical, end-to-end engineering work.

This repo captures one core idea: **don’t build roads across seas**. Understand the real problem, use existing paths, and ship coherent solutions.

## What’s in here

- `CLAUDE.md` — guiding philosophy, implementation standards, anti-patterns, and documentation expectations.
- `agents/start-session.md` — kickoff routine for a focused work session.
- `agents/think-first.md` — problem framing before implementation.
- `agents/review-approach.md` — quality review heuristics.
- `agents/refocus.md` — reset when execution drifts.
- `agents/build-connected.md` — integration-first implementation prompts.
- `agents/find-orphans.md` — detect dead or disconnected logic.
- `agents/coherence-validator.md` / `agents/validate-coherence.md` — consistency and cross-domain checks.

## How to use

1. Read `CLAUDE.md` first.
2. Pick one prompt from `agents/` that matches your current phase.
3. Work in small increments and keep integration checks tight.

## Why this exists

To keep engineering work practical, modern, and coherent—especially when moving fast.
